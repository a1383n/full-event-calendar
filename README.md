# vue-full-event-calendar


# usge
run this command in root to install packages in all workspaces
```
pnpm i
```
### vue2
run dev server for vue 2.6.11
```
pnpm run dev:vue-2
```
### vue3
run dev server for vue 3.26
```
pnpm run dev:vue-3
```

run this command in root to build Full Event Calender
```
pnpm run build
```
